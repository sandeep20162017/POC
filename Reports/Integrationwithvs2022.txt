Open your ASP.NET Core project in Visual Studio 2022.

Go to Tools > NuGet Package Manager > Manage NuGet Packages for Solution.

Search for and install the following packages:

Telerik.Reporting.Services.AspNetCore

Telerik.Reporting.OpenXmlRendering (optional, if you want to use OpenXML rendering)

Telerik.Reporting.XpsRendering (optional, if you want to use XPS rendering)

Step 2: Configure the Reporting REST Service
You need to configure a REST service in your ASP.NET Core application to serve the reports.

Create a new controller for the reporting service:

Right-click on the Controllers folder in your project and select Add > Controller.

Choose API Controller - Empty and name it something like ReportsController.
